ChemPal has keyboard shortcuts for the things you do most — jumping to search,
filtering results, clearing the cache, and more. Press **`Shift`+`?`** at any time
to open the in-app shortcut list, which now has a **search box** so you can find a
shortcut by name, category, or keys.

## Reading the keys

- **`mod`** means **`⌘` (Command)** on macOS and **`Ctrl`** on Windows/Linux.
- Combos joined with `+` are pressed **together** (a chord).
- Shortcuts don't fire while you're typing in a text field.
- Some shortcuts flash a brief confirmation in the status bar at the bottom.

## General

| Shortcut | What it does |
|----------|--------------|
| `Shift`+`?` | Show the list of keyboard shortcuts |

## Navigation

| Shortcut | What it does |
|----------|--------------|
| `⌘`+`S` (macOS) / `Ctrl`+`Shift`+`S` | Go to the search panel |
| `mod`+`H` | Open search history in the side drawer |

## Search

| Shortcut | What it does |
|----------|--------------|
| `mod`+`Shift`+`R` | Clear cache, clear results, and re-run the current search |
| `mod`+`.` | Abort the currently-running search |

## Results

| Shortcut | What it does |
|----------|--------------|
| `mod`+`F` | Focus the "Filter results…" box |
| `mod`+`Shift`+`F` | Toggle the per-column filter row |
| `mod`+`Shift`+`E` | Expand all rows in the results table |
| `mod`+`Shift`+`C` | Collapse all rows in the results table |
| `mod`+`Shift`+`↑` | Scroll the results table to the top |
| `mod`+`Shift`+`A` | Show all rows on one page (no pagination) |
| `mod`+`Shift`+`Backspace` | Clear all column filters |

## Sequential shortcuts

A few shortcuts are **sequences** — keys pressed one after another rather than all
at once (pause too long between keys and the sequence resets). ChemPal ships with a
small easter egg or two; go find them. 🙂
